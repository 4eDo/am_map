Запись в БД:

  1. id
  2. Тип (для поледующего выбора стиля обозначения)
    1. LAND_BORDER Граница страны
    2. LAND Страна
    3. REGION_BORDER Граница региона
    4. REGION Регион
    5. CAPITAL Столица
    6. BIG_TOWN Большой город
    7. TOWN Город
    8. LITTLE_TOWN Маленький город
    9. TOWN_MAP План города (у нас есть пара сгенерированных карт, если нет - будет пустое поле с текстовым описанием и чем-то ещё)
    10. FORT Крепость
    11. ABBEY Монастырь/скит/храм
    12. VILLAGE Деревня
    13. CAMP Становище и т.п., что обозначается палаткой
    14. HYDROLOGY Океан/Море/Залив
    15. RIVER Река
    16. FOREST Лес
    17. MOUNTAINS Горы
    18. ISLAND Остров
    19. OTHER Иной объект (то, для чего не нашлось иконок, будет подтягиваться отдельной ссылкой н картинку)
  3. Список "
    idРодителя|координатаX|координатаY|допСтиль###idРодителя|координатаX|координатаY|допСтиль###...
    " для вывода объекта на разные представления карты, где idРодителя - идентификатор родительского объекта (на котором этот отображается), координатаX - значение margin-left от родителя в пикселях, координатаY - значение margin-top от родителя в пикселях, допСтиль - отдельный стиль для объекта (например, поворот, меньший размер и т.п.), ### - разделитель между разными записями
  4. Название
  5. Описание
  6. Дополнительная информация
  7. Ссылка на картинку (только для границ, стран, регионов, отдельных объектов и т.п., остальному не нужно)

В итоге будет отражено:

На общей карте появятся границы Атравана (ОЪЕКТ А), при клике на название страны (ОЪЕКТ Б) в списке стран подсвечивающиеся зелёным Амалиррским светом. Также будет выведено название страны(ОЪЕКТ Б) по координатам из списка координат для id ОБЪЕКТА А. Будет выводиться столица с названием (ОЪЕКТ В). При этом в отдельном блоке будет появляться краткая информация о стране (ОЪЕКТ Б) и столице (ОЪЕКТ В)

При открытии карты самого Атравана (ОЪЕКТ Б) в инфоблок будет выведено название и описание страны (ОЪЕКТ Б), название столицы (ОЪЕКТ В). Также на карту будет выведена столица с названием (ОЪЕКТ В).
